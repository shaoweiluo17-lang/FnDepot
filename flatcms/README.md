# FlatCMS-极速建站

FlatCMS 是一个轻量化网站与内容管理应用，支持三级目录导航、文章发布、脚本分享与站点迁移。

## 应用信息

- 应用名：FlatCMS-极速建站
- 包名：`flatcms`
- 当前版本：`1.0.5`
- 发布者：青芷是大佬
- 原维护站点：<http://qdnas.icu/>

## 简介

极速建站创建属于你自己的网站，支持手机端操作、CMS 内容分发、Markdown 导入与 ZIP 离线备份。内置多重安全防护、多语言支持（中英日韩繁）、SQLite 数据库与双层缓存。

默认用户名密码：`admin/admin`。

## 安装说明

从 FnDepot 安装后即可使用。当前仓库内提供 `flatcms.fpk` 安装包。
