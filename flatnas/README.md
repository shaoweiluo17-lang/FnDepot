# FlatNas导航

FlatNas 是一个支持多端同步的个人导航页与仪表盘系统，适合统一管理常用信息与日常工作入口。

## 应用信息

- 应用名：FlatNas导航
- 包名：`flatnas`
- 当前版本：`1.1.3-dev3`
- 发布者：青芷是大佬
- 原维护站点：<https://flatnas.top/>

## 简介

FlatNas 提供个人导航页、文件传输助手与备忘同步能力，并支持 RSS、日历、时钟、天气、待办等组件，在手机、电脑和平板间保持数据同步。

默认端口：`23000`，默认密码：`admin`。

## 安装说明

从 FnDepot 安装后即可使用。当前仓库内提供 `flatnas.fpk` 安装包。
